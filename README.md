# GEDCOM Merger v2.4.0

Application de fusion de doublons GEDCOM.

## Installation

```bash
npm install
npm run dev
```

## Déploiement

Déployé sur Netlify automatiquement.

## Fonctionnalités

- Détection de doublons
- Fusion intelligente
- Normalisation des lieux
- Fusion guidée contextuelle v2.4.0
